# App 端用户中心缺失功能清单 + 推荐接口草案

## 1. 范围说明

本文只讨论 `AppUser` 在 app 端“用户中心”视角下缺失的能力，不包含以下内容：

- 验证码相关能力
- 设备列表、设备管理、踢设备下线
- 依赖验证码才能安全落地的手机号、邮箱换绑流程

目标是明确：

- 现在 app 端用户中心已经有什么
- 还缺什么
- 推荐怎么设计接口，避免和现有模块重复

## 2. 当前结论

用户中心侧建议只保留以下能力：

- 个人资料查看与编辑
- 论坛资料查看与编辑
- 用户中心首页聚合
- 用户状态摘要
- 资产摘要
- 成长摘要
- 积分统计与流水
- 经验统计与流水
- 我的徽章

不建议新增设备管理相关接口，后续也不再纳入用户中心方案。
