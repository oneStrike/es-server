# 事件到任务映射检查清单

## 1. 适用范围

本清单用于在新增或改造“事件驱动任务”前，检查某个业务事件是否已经具备被任务系统消费的前置条件。

适用对象：

- `P1-01` Producer 迁移评审
- `P1-02` 事件驱动任务接入评审
- 新增任务模板配置前的事件映射确认
